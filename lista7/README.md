## Lista VII

### Exercício 1 - Média Altura
>Fazer um programa para ler um número inteiro N e a altura e N pessoas. Armazene as N alturas em um vetor. Em seguida, mostrar a altura média das pessoas.
>
>Exemplo:
>| Entrada | Saída               |
>|---------|---------------------|
>| 3       | Altura média = 1.69 |
>| 1.72    |                     |
>| 1.56    |                     |
>| 1.80    |                     |

### Exercício 3 - Negativos
>Faça um programa que leia um número inteiro positivo N (máximo = 10) e depois N números inteiros
e armazene-os em um vetor. Em seguida, mostrar na tela todos os números negativos lidos. 
>
>Exemplo:
>| Entrada                           |    |        Saída       |
>|-----------------------------------|----|:------------------:|
>| Quantos números você vai digitar? | 6  | Números negativos: |
>| Digite um número:                 | 8  |         -2         |
>| Digite um número:                 | -2 |         -3         |
>| Digite um número:                 | 9  |         -7         |
>| Digite um número:                 | 10 |                    |
>| Digite um número:                 | -3 |                    |
>| Digite um número:                 | -7 |                    |

### Exercício 9 - Abaixo da Média

>| Altura da 2ª pessoas:            | 1.83 |          Número de homens = 2         |
>| Gênero da 2ª pessoa:             | M    |                                       |

>| Gênero da 3ª pessoa:             | M    |                                       |
>| Altura da 4ª pessoas:            | 1.61 |                                       |
>| Gênero da 4ª pessoa:             | F    |                                       |
>| Altura da 5ª pessoas:            | 1.75 |                                       |
>| Gênero da 5ª pessoa:             | F    |                                       |
